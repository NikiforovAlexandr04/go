Go - игра целью которой является захват територии и камней противника. Два игрока ставят по очереди камни на доску, выигрывает тот кто окружил больше територии и камней противника своими камнями. Игра заканчивается когда оба игрока нажимают "пас".
Клавиша 'T' показывает таблицу лидеров игры.

main.py - главный файл для запуска игры.

pictures - папка с картинками
buttons_interface - графика кнопок выбора
field_interface - графика поля
pass interface - графика кнопок паса и игры
stone - элемент камня
field - главный файл в котором происходят игровые события
test - тесты
